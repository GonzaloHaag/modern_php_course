# PATRONES DE DISEÑO 

## Un patron de diseño es una tecnica para resolver problemas comunes habituales 

## Existen 3 tipos bases 
1. Tipo comportamiento: Nos dara una forma base para organizar las clases e interfaces para comportamientos especificos  
2. Tipo creacionales: Estrategias de creacion de objetos dependiendo sus caracteristicas
3. Tipo estructurales: Nos dara una forma de estructuracion de como heredemos nuestras clases e implementaciones (jerarquias)
